Overview
The AI-Based Student Skill Gap Analyzer is a Natural Language Processing (NLP) and Machine Learning based project designed to help students identify the gap between their current skills and the skills required by industry job roles.
The system analyzes a student's resume and compares it with job descriptions to determine how well the candidate's skills match the job requirements. It calculates a skill match score, identifies missing skills, and provides insights that can help students improve their career readiness.

Problem Statement
Many students are unsure whether their skills align with the requirements of industry roles. While job descriptions clearly mention required technologies and competencies, students often find it difficult to evaluate how well their resumes match those expectations.
This project aims to bridge that gap by building an intelligent system that automatically analyzes resumes and job descriptions using NLP and Machine Learning techniques.

Key Features
Extracts text from resume PDFs
Performs NLP preprocessing (text cleaning, stopword removal)
Converts text into numerical vectors using TF-IDF
Computes similarity between resume and job description using Cosine Similarity
Calculates a skill match score
Identifies missing skills
Provides insights to help students improve their skillset
